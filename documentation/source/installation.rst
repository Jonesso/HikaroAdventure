How to install
===========================================

The fastest way to run the game is to run it in installed project. This should be enough for quickstart:

.. code-block:: bash

   $ git clone https://github.com/Jonesso/HikaroAdventure
   $ cd HikaroAdventure
   $ pip install -r requirements.txt
   $ python main.py
